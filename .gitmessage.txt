# 타입 가이드
# feat : 새로운 기능을 추가하는 경우
# fix : 버그를 고친 경우
# docs : 문서를 수정한 경우
# style : 코드 포맷 변경, 세미콜론 누락 등 코드 변경이 없는 경우
